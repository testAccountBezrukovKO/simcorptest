HOWTO:
List types: SingleLinkedList and DoubleLinkedList
SingleLinkedList have next functionality:
- Add : add new item to list
- Remove : remove item from list
- GetNodeByValue: get list node by value
- GetListValues : get all list values

DoubleLinkedList have next functionality:
- Add : add new item to list
- Remove : remove item from list
- GetNodeByValue: get list node by value
- GetListValues : get all list values
